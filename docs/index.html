About this book

Learning Statistics with CogStat covers the contents of an introductory statistics class, as typically taught to undergraduate psychology students based on Danielle Navarro’s original Learning Statistics with R (Version 0.63) with a focus on automatic statistical analysis performed in CogStat.

Cogstat is a statistics software written in Python by Attila Krajcsi4 and developed with the help of supporters. Its distinct advantage is automatic hypothesis test selection and chart creation with an APA style output to suit the needs of psychology researchers and students.
